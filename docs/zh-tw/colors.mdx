我們完整的色彩調色盤應用於圖形與插圖中，探索獨特且現代的色彩組合以及單色對比。它可用於突顯特定主題或強調重點。

F5 調色盤使用七種主要品牌色彩加上黑色與白色，每種顏色各有四個淡色／深色變體，共計 **45 種顏色**。本頁面是主題開發期間將品牌色彩對應至 Starlight CSS 自訂屬性的唯一可信來源。

## 對應至 Starlight CSS 自訂屬性

Starlight 公開了 `--sl-color-*` 變數供主題使用。以下是 F5 調色盤對應至 Starlight 語意色彩插槽的對應關係，實作於 `styles/custom.css`。

> **Starlight 主題設定運作方式**：在 Starlight 中，**深色模式是裸 `:root` 的預設值**——不存在 `data-theme="dark"` 選擇器。淺色模式使用 `:root[data-theme='light']`。灰階變數是*語意性的*，而非依亮度排序：`gray-1` 永遠是「主要文字」，而 `gray-6`／`gray-7` 永遠是「輕微背景」，不論何種模式皆然。其實際亮度值會在模式之間互換。同樣地，`accent-high` 永遠是高可見度強調文字色彩，而 `accent-low` 永遠是輕微強調背景——其亮度在模式之間反轉。

```css
/* 深色模式 — Starlight 預設 (:root) */
:root {
```

### 對比度驗證（WCAG AA）

**深色模式**（文字於 `#000000` 頁面背景上）：
- 內文文字 `#cccccc` → **16.3:1**（AAA）
- 暗淡文字 `#999999` → **10:1**（AAA）
- 強調連結 `#f06680` → **5.6:1**（AA）

**淺色模式**（文字於 `#ffffff` 頁面背景上）：
- 內文文字 `#343434` → **11.6:1**（AAA）
- 暗淡文字 `#666666` → **5.7:1**（AA）
- 強調連結 `#e4002b` → **4.6:1**（AA）

## 色彩對比

為確保所有人都能存取並理解我們的內容，請使用符合無障礙標準的背景與文字色彩組合。選擇大型文字對比比率至少為 3:1、一般文字至少為 4.5:1 的色彩配對。

所有色彩配對必須符合 **WCAG AA** 對比要求：

- **一般文字**（< 18pt／14pt 粗體）：最低對比比率 **4.5:1**
- **大型文字**（≥ 18pt／14pt 粗體）：最低對比比率 **3:1**
- **UI 元件與圖形物件**：最低對比比率 **3:1**
