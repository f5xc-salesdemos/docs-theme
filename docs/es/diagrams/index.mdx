El tema de documentación incluye una integración con Mermaid y paquetes de iconos personalizados para la creación de diagramas de arquitectura. Explore el catálogo a continuación para encontrar ejemplos para su caso de uso.

## Primeros pasos

Consulte la página de [Instrucciones](/diagrams/instructions/) para referencia de sintaxis, paquetes de iconos disponibles y detalles de estilos CSS.

## Tipos de diagramas

### Fundamentos

- [Diagramas básicos](/diagrams/basic-diagrams/) — Ejemplos de diagramas de flujo, secuencia, circular, de clase, de estado y de Gantt

### Proveedores de nube

- [AWS](/diagrams/aws/) — Arquitecturas de VPC, subredes, ALB, EC2, EKS, Lambda y S3
- [Azure](/diagrams/azure/) — VNet, App Gateway, VM Scale Sets, AKS y topología Hub-Spoke
- [GCP](/diagrams/gcp/) — VPC, GKE, Cloud Functions, Cloud SQL y balanceador de carga global
- [Multi-Cloud](/diagrams/multi-cloud/) — Topología entre nubes, balanceo de carga global, conexión multinube de F5 XC

### Seguridad

- [WAF](/diagrams/waf/) — Pipeline de inspección del Firewall de aplicaciones web (WAF) y protección OWASP
- [DDoS](/diagrams/ddos/) — Arquitectura de mitigación de DDoS y servicios de limpieza de tráfico
- [Protección de API](/diagrams/api-protection/) — Seguridad de gateway de API, descubrimiento de API ocultas, limitación de velocidad
- [Defensa Bot](/diagrams/bot-defense/) — Pipeline de detección de bots, defensa contra credential stuffing
- [Zero Trust](/diagrams/zero-trust/) — Arquitectura ZTNA, verificación de identidad, microsegmentación

### Infraestructura

- [DNS](/diagrams/dns/) — Resolución DNS, GSLB, gestión de DNS en F5 XC
- [Kubernetes](/diagrams/kubernetes/) — Controladores de Ingress, service mesh, redes de pods
- [Terraform](/diagrams/terraform/) — Automatización de IaC con HashiCorp Consul, Vault y Nomad

### Productos F5

- [Marca F5](/diagrams/f5-brand/) — Iconos de servicios de F5 XC, línea de productos NGINX, iconos de BIG-IP
- [F5 Distributed Cloud](/diagrams/f5xc-diagrams/) — XC WAAP, defensa bot, Redes multinube, DDoS, DNS, Seguridad de API utilizando iconos de marca F5
