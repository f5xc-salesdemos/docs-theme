Le thème de documentation inclut une intégration Mermaid avec des paquets d'icônes personnalisés pour la création de diagrammes d'architecture. Parcourez le catalogue ci-dessous pour trouver des exemples adaptés à votre cas d'utilisation.

## Premiers pas

Consultez la page [Instructions](/diagrams/instructions/) pour la référence de syntaxe, les paquets d'icônes disponibles et les détails de style CSS.

## Types de diagrammes

### Fondamentaux

- [Diagrammes de base](/diagrams/basic-diagrams/) — Exemples de diagrammes de flux, de séquence, circulaires, de classes, d'états et de diagrammes de Gantt

### Fournisseurs cloud

- [AWS](/diagrams/aws/) — Architectures VPC, sous-réseaux, ALB, EC2, EKS, Lambda et S3
- [Azure](/diagrams/azure/) — VNet, passerelle d'application, groupes de machines virtuelles identiques, AKS et topologie Hub-Spoke
- [GCP](/diagrams/gcp/) — VPC, GKE, Cloud Functions, Cloud SQL et équilibrage de charge global
- [Multi-Cloud](/diagrams/multi-cloud/) — Topologie multi-cloud, équilibrage de charge global, connexion multi-cloud F5 XC

### Sécurité

- [WAF](/diagrams/waf/) — Pipeline d'inspection du pare-feu applicatif (WAF) et protection OWASP
- [DDoS](/diagrams/ddos/) — Architecture de mitigation DDoS et services de nettoyage
- [Protection des API](/diagrams/api-protection/) — Sécurité de la passerelle API, découverte des API fantômes, limitation du débit
- [Défense Bot](/diagrams/bot-defense/) — Pipeline de détection des bots, défense contre le bourrage d'identifiants
- [Zéro confiance](/diagrams/zero-trust/) — Architecture ZTNA, vérification d'identité, micro-segmentation

### Infrastructure

- [DNS](/diagrams/dns/) — Résolution DNS, GSLB, gestion DNS F5 XC
- [Kubernetes](/diagrams/kubernetes/) — Contrôleurs Ingress, maillage de services, réseau de pods
- [Terraform](/diagrams/terraform/) — Automatisation IaC avec HashiCorp Consul, Vault et Nomad

### Produits F5

- [Marque F5](/diagrams/f5-brand/) — Icônes de services F5 XC, gamme de produits NGINX, icônes BIG-IP
- [F5 Distributed Cloud](/diagrams/f5xc-diagrams/) — XC WAAP, défense bot, réseau multi-cloud, Protection DDoS, DNS, Sécurité des API utilisant les icônes de marque F5
