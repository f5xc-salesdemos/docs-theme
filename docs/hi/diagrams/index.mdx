डॉक्स थीम में कस्टम आइकन पैक के साथ एक Mermaid इंटीग्रेशन शामिल है जो आर्किटेक्चर डायग्राम बनाने के लिए उपयोगी है। अपने उपयोग के मामले के लिए उदाहरण खोजने हेतु नीचे दी गई कैटलॉग देखें।

## शुरुआत करना

सिंटैक्स संदर्भ, उपलब्ध आइकन पैकेज और CSS स्टाइलिंग विवरण के लिए [निर्देश](/diagrams/instructions/) पृष्ठ देखें।

## डायग्राम के प्रकार

### मूल बातें

- [बेसिक डायग्राम](/diagrams/basic-diagrams/) — फ्लोचार्ट, सीक्वेंस, पाई, क्लास, स्टेट और गैंट चार्ट के उदाहरण

### क्लाउड प्रदाता

- [AWS](/diagrams/aws/) — VPC, सबनेट, ALB, EC2, EKS, Lambda और S3 आर्किटेक्चर
- [Azure](/diagrams/azure/) — VNet, App Gateway, VM Scale Sets, AKS और Hub-Spoke टोपोलॉजी
- [GCP](/diagrams/gcp/) — VPC, GKE, Cloud Functions, Cloud SQL और Global LB
- [मल्टी-क्लाउड](/diagrams/multi-cloud/) — क्रॉस-क्लाउड टोपोलॉजी, ग्लोबल लोड बैलेंसिंग, F5 XC मल्टी-क्लाउड कनेक्ट

### सुरक्षा

- [WAF](/diagrams/waf/) — वेब एप्लिकेशन फ़ायरवॉल इंस्पेक्शन पाइपलाइन और OWASP सुरक्षा
- [DDoS](/diagrams/ddos/) — DDoS सुरक्षा आर्किटेक्चर और स्क्रबिंग सेवाएँ
- [API सुरक्षा](/diagrams/api-protection/) — API गेटवे सुरक्षा, शैडो API डिस्कवरी, रेट लिमिटिंग
- [Bot उन्नत रक्षा](/diagrams/bot-defense/) — Bot डिटेक्शन पाइपलाइन, क्रेडेंशियल स्टफिंग डिफेंस
- [ज़ीरो ट्रस्ट](/diagrams/zero-trust/) — ZTNA आर्किटेक्चर, पहचान सत्यापन, माइक्रो-सेगमेंटेशन

### इन्फ्रास्ट्रक्चर

- [DNS](/diagrams/dns/) — DNS रिज़ॉल्यूशन, GSLB, F5 XC DNS प्रबंधन
- [Kubernetes](/diagrams/kubernetes/) — इंग्रेस कंट्रोलर, सर्विस मेश, पॉड नेटवर्किंग
- [Terraform](/diagrams/terraform/) — HashiCorp Consul, Vault और Nomad के साथ IaC स्वचालन

### F5 उत्पाद

- [F5 ब्रांड](/diagrams/f5-brand/) — F5 XC सर्विस आइकन, NGINX प्रोडक्ट लाइन, BIG-IP आइकन
- [F5 Distributed Cloud](/diagrams/f5xc-diagrams/) — F5 ब्रांड आइकन का उपयोग करते हुए XC WAAP, Bot उन्नत रक्षा, मल्टी-क्लाउड नेटवर्किंग, DDoS सुरक्षा, DNS, API सुरक्षा
