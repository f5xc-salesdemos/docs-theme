हमारे रंग पैलेट की पूरी श्रृंखला ग्राफिक्स और चित्रणों में उपयोग की जाती है, जो अनूठे और आधुनिक संयोजनों के साथ-साथ मोनोक्रोमेटिक कंट्रास्ट को भी explore करती है। इसका उपयोग किसी विशिष्ट विषय या जोर देने योग्य बिंदु को उजागर करने के लिए किया जा सकता है।

F5 पैलेट में सात प्राथमिक ब्रांड रंगों के साथ काले और सफेद रंग भी शामिल हैं, जिनमें से प्रत्येक के चार टिंट/शेड वेरिएंट हैं, जो कुल **45 रंग** बनाते हैं। यह पृष्ठ थीम विकास के दौरान ब्रांड रंगों को Starlight CSS कस्टम प्रॉपर्टीज़ में मैप करने का एकमात्र आधिकारिक स्रोत है।

## द्वितीयक रंग (टिंट और शेड)

प्रत्येक प्राथमिक रंग के चार वेरिएंट हैं जो सबसे हल्के (Level 1) से सबसे गहरे (Level 4) के क्रम में हैं।

Brand Center द्वितीयक रंगों को LESS वेरिएबल नाम प्रदान करता है। Cloud Red और Tangerine वेरिएंट के अद्वितीय, सार्थक नाम हैं। अन्य सभी रंग समूह समान placeholder नाम (`wewak`, `froly`, `carmine`, `venetian-red`) प्रदर्शित करते हैं — ये ब्रांड प्लेटफ़ॉर्म के टेम्प्लेट डिफ़ॉल्ट हैं और पूर्णता के लिए नीचे शामिल किए गए हैं, लेकिन इन्हें विहित पहचानकर्ता के रूप में नहीं माना जाना चाहिए।

## Starlight CSS कस्टम प्रॉपर्टीज़ में मैपिंग

Starlight थीमिंग के लिए `--sl-color-*` वेरिएबल्स उजागर करता है। नीचे F5 पैलेट से Starlight के semantic color slots में मैपिंग दी गई है, जो `styles/custom.css` में कार्यान्वित की गई है।

> **Starlight थीमिंग कैसे काम करती है**: Starlight में, **डार्क मोड bare `:root` डिफ़ॉल्ट है** — कोई `data-theme="dark"` सिलेक्टर नहीं है। लाइट मोड `:root[data-theme='light']` का उपयोग करता है। gray scale वेरिएबल्स *semantic* हैं, brightness-ordered नहीं: `gray-1` हमेशा "primary text" होता है और `gray-6`/`gray-7` हमेशा "subtle backgrounds" होते हैं, चाहे मोड कोई भी हो। उनके वास्तविक brightness मान मोड के बीच swap हो जाते हैं। इसी प्रकार, `accent-high` हमेशा high-visibility accent text color होता है और `accent-low` हमेशा subtle accent background होता है — उनकी brightness मोड के बीच उलट जाती है।

```css
/* Dark mode — Starlight default (:root) */
:root {
  --sl-color-white: #ffffff;      /* Heading text (brightest) */
  --sl-color-gray-1: #f5f5f5;    /* Primary text — White Level 2 */
  --sl-color-gray-2: #cccccc;    /* Body text — White Level 4 */
  --sl-color-gray-3: #999999;    /* Dim/muted text — Black Level 1 */
  --sl-color-gray-4: #666666;    /* Borders — Black Level 2 */
  --sl-color-gray-5: #343434;    /* Inline code bg — Black Level 3 */
  --sl-color-gray-6: #222222;    /* Sidebar bg — Black Level 4 */
  --sl-color-black: #000000;     /* Page background */
  --sl-color-accent-low: #720016;   /* Subtle accent bg — F5 Red Level 4 */
  --sl-color-accent: #f06680;       /* Links, active — F5 Red Level 2 */
  --sl-color-accent-high: #f7b2bf;  /* Accent text — F5 Red Level 1 */
}

/* Light mode — inverted gray scale and accent brightness */
:root[data-theme='light'] {
  --sl-color-white: #222222;      /* Heading text (darkest) — Black Level 4 */
  --sl-color-gray-1: #222222;    /* Primary text — Black Level 4 */
  --sl-color-gray-2: #343434;    /* Body text — Black Level 3 */
  --sl-color-gray-3: #666666;    /* Dim text — Black Level 2 */
  --sl-color-gray-4: #999999;    /* Borders — Black Level 1 */
  --sl-color-gray-5: #cccccc;    /* Light borders — White Level 4 */
  --sl-color-gray-6: #f5f5f5;    /* Sidebar bg — White Level 2 */
  --sl-color-gray-7: #faf9f7;    /* Nav bg — White Level 1 */
  --sl-color-black: #ffffff;     /* Page background */
  --sl-color-accent-low: #f7b2bf;   /* Subtle accent bg — F5 Red Level 1 (inverted) */
  --sl-color-accent: #e4002b;       /* Links, active — F5 Red */
  --sl-color-accent-high: #720016;  /* Accent text — F5 Red Level 4 (inverted) */
}
```

### कंट्रास्ट सत्यापन (WCAG AA)

**डार्क मोड** (`#000000` पृष्ठ पृष्ठभूमि पर टेक्स्ट):
- Body text `#cccccc` → **16.3:1** (AAA)
- Dim text `#999999` → **10:1** (AAA)
- Accent link `#f06680` → **5.6:1** (AA)

**लाइट मोड** (`#ffffff` पृष्ठ पृष्ठभूमि पर टेक्स्ट):
- Body text `#343434` → **11.6:1** (AAA)
- Dim text `#666666` → **5.7:1** (AA)
- Accent link `#e4002b` → **4.6:1** (AA)

## रंग कंट्रास्ट

यह सुनिश्चित करने के लिए कि हर कोई हमारे संचार को एक्सेस कर सके और समझ सके, पृष्ठभूमि और टेक्स्ट रंग संयोजनों का उपयोग करें जो एक्सेसिबिलिटी मानकों को पूरा करते हों। बड़े टेक्स्ट के लिए न्यूनतम 3:1 और सामान्य टेक्स्ट के लिए 4.5:1 कंट्रास्ट अनुपात वाले रंग युग्मों का चयन करें।

सभी रंग युग्मों को **WCAG AA** कंट्रास्ट आवश्यकताओं को पूरा करना चाहिए:

- **सामान्य टेक्स्ट** (< 18pt / 14pt bold): न्यूनतम **4.5:1** कंट्रास्ट अनुपात
- **बड़ा टेक्स्ट** (≥ 18pt / 14pt bold): न्यूनतम **3:1** कंट्रास्ट अनुपात
- **UI घटक और ग्राफिकल ऑब्जेक्ट**: न्यूनतम **3:1** कंट्रास्ट अनुपात

## थीम जाँच

- `.swatch` बॉर्डर `--sl-color-gray-5` का उपयोग करता है
- `.swatch-label` पृष्ठभूमि `--sl-color-gray-6` का उपयोग करती है

- थीम टॉगल करने पर Badge टोकन swatches अपडेट होते हैं
