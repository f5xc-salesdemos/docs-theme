カラーパレットの全範囲はグラフィックやイラストに使用され、ユニークでモダンな組み合わせのほか、モノクロマティックなコントラストも探求しています。特定のトピックや強調したいポイントを際立たせるために活用できます。

F5 パレットは、7 つのプライマリブランドカラーにブラックとホワイトを加えた構成で、それぞれに 4 つのティント／シェードバリアントを持ち、合計 **45 色** となっています。このページは、テーマ開発時にブランドカラーを Starlight CSS カスタムプロパティにマッピングするための唯一の信頼できる情報源です。

## Starlight CSS カスタムプロパティへのマッピング

Starlight はテーマ設定用に `--sl-color-*` 変数を公開しています。以下は、F5 パレットから Starlight のセマンティックカラースロットへのマッピングで、`styles/custom.css` に実装されています。

> **Starlight テーマの仕組み**: Starlight では、**ダークモードが `:root` のデフォルト**です — `data-theme="dark"` セレクターはありません。ライトモードは `:root[data-theme='light']` を使用します。グレースケール変数は*セマンティック*であり、明度順ではありません。`gray-1` は常に「プライマリテキスト」を表し、`gray-6`/`gray-7` は常に「サブトルな背景」を表します（モードに関わらず）。実際の明度値はモード間で入れ替わります。同様に、`accent-high` は常に高視認性のアクセントテキストカラーであり、`accent-low` は常にサブトルなアクセント背景です — モード間で明度が反転します。

```css
/* ダークモード — Starlight デフォルト (:root) */
:root {
  --sl-color-white: #ffffff;      /* 見出しテキスト（最も明るい） */
  --sl-color-gray-1: #f5f5f5;    /* プライマリテキスト — White レベル 2 */
  --sl-color-gray-2: #cccccc;    /* ボディテキスト — White レベル 4 */
  --sl-color-gray-3: #999999;    /* 薄い/ミュートテキスト — Black レベル 1 */
  --sl-color-gray-4: #666666;    /* ボーダー — Black レベル 2 */
  --sl-color-gray-5: #343434;    /* インラインコード背景 — Black レベル 3 */
  --sl-color-gray-6: #222222;    /* サイドバー背景 — Black レベル 4 */
  --sl-color-black: #000000;     /* ページ背景 */
  --sl-color-accent-low: #720016;   /* サブトルなアクセント背景 — F5 Red レベル 4 */
  --sl-color-accent: #f06680;       /* リンク、アクティブ — F5 Red レベル 2 */
  --sl-color-accent-high: #f7b2bf;  /* アクセントテキスト — F5 Red レベル 1 */
}

/* ライトモード — 反転したグレースケールとアクセントの明度 */
:root[data-theme='light'] {
  --sl-color-white: #222222;      /* 見出しテキスト（最も暗い） — Black レベル 4 */
  --sl-color-gray-1: #222222;    /* プライマリテキスト — Black レベル 4 */
  --sl-color-gray-2: #343434;    /* ボディテキスト — Black レベル 3 */
  --sl-color-gray-3: #666666;    /* 薄いテキスト — Black レベル 2 */
  --sl-color-gray-4: #999999;    /* ボーダー — Black レベル 1 */
  --sl-color-gray-5: #cccccc;    /* 薄いボーダー — White レベル 4 */
  --sl-color-gray-6: #f5f5f5;    /* サイドバー背景 — White レベル 2 */
  --sl-color-gray-7: #faf9f7;    /* ナビゲーション背景 — White レベル 1 */
  --sl-color-black: #ffffff;     /* ページ背景 */
  --sl-color-accent-low: #f7b2bf;   /* サブトルなアクセント背景 — F5 Red レベル 1（反転） */
  --sl-color-accent: #e4002b;       /* リンク、アクティブ — F5 Red */
  --sl-color-accent-high: #720016;  /* アクセントテキスト — F5 Red レベル 4（反転） */
}
```

### コントラスト検証（WCAG AA）

**ダークモード**（`#000000` のページ背景上のテキスト）:
- ボディテキスト `#cccccc` → **16.3:1**（AAA）
- 薄いテキスト `#999999` → **10:1**（AAA）
- アクセントリンク `#f06680` → **5.6:1**（AA）

**ライトモード**（`#ffffff` のページ背景上のテキスト）:
- ボディテキスト `#343434` → **11.6:1**（AAA）
- 薄いテキスト `#666666` → **5.7:1**（AA）
- アクセントリンク `#e4002b` → **4.6:1**（AA）

## カラーコントラスト

すべてのユーザーがコミュニケーションにアクセスして理解できるよう、アクセシビリティ基準を満たす背景色とテキスト色の組み合わせを使用してください。大きなテキストには最小コントラスト比 3:1、通常のテキストには 4.5:1 を持つカラーペアを選択してください。

すべてのカラーペアリングは **WCAG AA** コントラスト要件を満たす必要があります：

- **通常のテキスト**（18pt 未満 / 14pt 未満の太字）: 最小 **4.5:1** コントラスト比
- **大きなテキスト**（18pt 以上 / 14pt 以上の太字）: 最小 **3:1** コントラスト比
- **UI コンポーネントおよびグラフィカルオブジェクト**: 最小 **3:1** コントラスト比

### 推奨される安全なペアリング

| 前景色 | 背景色 | コントラスト比 | 合格 |
|------------|------------|----------------|--------|
| White `#ffffff` | F5 Red `#e4002b` | 4.6:1 | AA |
| White `#ffffff` | River `#0e41aa` | 6.4:1 | AA / AAA |
| White `#ffffff` | Bay `#0072b0` | 4.6:1 | AA |
| White `#ffffff` | Jade `#009639` | 4.2:1 | AA（大きなテキスト） |
| White `#ffffff` | Eggplant `#62228b` | 6.5:1 | AA / AAA |
| White `#ffffff` | Raspberry `#ab2782` | 4.7:1 | AA |
| Black `#000000` | Tangerine `#f29a36` | 7.8:1 | AA / AAA |
| Black `#000000` | F5 Red レベル 1 `#f7b2bf` | 10.1:1 | AA / AAA |
| Black `#222222` | White レベル 1 `#faf9f7` | 15.8:1 | AA / AAA |

> **ヒント**: 新しいカラーの組み合わせを作成する際は、[WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) などのツールで必ずコントラスト比を確認してください。

## テーマチェック

- `.swatch` のボーダーは `--sl-color-gray-5` を使用
- `.swatch-label` の背景は `--sl-color-gray-6` を使用
- すべての `--f5-*` スウォッチは実際のカラーを表示（CSS カスタムプロパティは `custom.css` から解決）
- カラースウォッチはライトモードとダークモードの両方で表示可能
- バッジトークンのスウォッチはテーマ切り替え時に更新
