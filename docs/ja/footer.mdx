テーマはデフォルトの Starlight フッターを、標準フッターコンテンツの下にソーシャルメディアリンクを追加するカスタム `Footer.astro` コンポーネントに置き換えます。

## コンポーネントパターン

```astro
---
import Default from '@astrojs/starlight/components/Footer.astro';
---

<Default {...Astro.props}><slot /></Default>

<div class="social-links">
  <!-- social link icons -->
</div>
```

このパターンはデフォルトのフッターを完全に置き換えるのではなく、**拡張**します。`<Default {...Astro.props}><slot /></Default>` は元の Starlight フッター（前へ/次へナビゲーション）をレンダリングし、その下にソーシャルリンクが追加されます。

## 登録方法

フッターのオーバーライドは、Starlight プラグインによって `index.ts` に登録されます。`config:setup` フック中に `Footer` コンポーネントのオーバーライドを設定します。

```ts
updateConfig({
  components: {
    ...config.components,
    Footer: '@f5-sales-demo/docs-theme/components/Footer.astro',
    // ... other component overrides
  },
});
```

コンテンツリポジトリはフッターを手動で登録する必要はありません。テーマプラグインがロードされると自動的に適用されます。

## CSS スタイリング

```css
.social-links {
  display: flex;
  justify-content: flex-end;
  align-items: center;
  gap: 1.25rem;
  padding: 1rem 0 0.5rem;
}

.social-links a {
  display: inline-flex;
  transition: opacity 0.2s ease;
}

.social-links a:hover {
  opacity: 0.7;
}
```

リンクはフレックス行内に右揃えで配置され、ホバー時にわずかな透明度効果が適用されます。

## カスタマイズ

### URL の変更

`@f5-sales-demo/docs-theme` パッケージ内の `components/Footer.astro` にある対応する `<a>` タグの `href` 属性を編集します。

### リンクの追加

`.social-links` div 内に 24x24 の SVG アイコンを含む新しい `<a>` 要素を追加します。既存のパターンに従ってください：

```html
<a href="https://example.com" target="_blank" rel="noopener noreferrer" aria-label="Platform Name">
  <svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="#COLOR">
    <path d="..."/>
  </svg>
</a>
```

### リンクの削除

削除したいプラットフォームの `<a>...</a>` ブロック全体を削除します。

### アイコンカラーの変更

`<svg>` 要素の `fill` 属性を変更します。テキストカラーを継承するには `currentColor` を使用し、ブランドカラーには特定の16進数値を使用します。

### レイアウトの変更

`<style>` ブロック内の `.social-links` CSS を変更します：
- **中央揃え**: `justify-content` を `center` に変更
- **間隔を広げる**: `gap` の値を増加
- **左揃え**: `justify-content` を `flex-start` に変更
