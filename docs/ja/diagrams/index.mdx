ドキュメントテーマには、アーキテクチャダイアグラムを構築するためのカスタムアイコンパックを備えた Mermaid インテグレーションが含まれています。以下のカタログを参照して、ユースケースに合ったサンプルを見つけてください。

## はじめに

構文リファレンス、利用可能なアイコンパッケージ、および CSS スタイリングの詳細については、[手順](/diagrams/instructions/)ページを参照してください。

## ダイアグラムの種類

### 基礎

- [基本ダイアグラム](/diagrams/basic-diagrams/) — フローチャート、シーケンス、円グラフ、クラス、状態、およびガントチャートのサンプル

### クラウドプロバイダー

- [AWS](/diagrams/aws/) — VPC、サブネット、ALB、EC2、EKS、Lambda、および S3 アーキテクチャ
- [Azure](/diagrams/azure/) — VNet、App Gateway、VM スケールセット、AKS、およびハブスポークトポロジー
- [GCP](/diagrams/gcp/) — VPC、GKE、Cloud Functions、Cloud SQL、およびグローバル LB
- [マルチクラウド](/diagrams/multi-cloud/) — クロスクラウドトポロジー、グローバルロードバランシング、F5 XC マルチクラウドコネクト

### セキュリティ

- [WAF](/diagrams/waf/) — Web アプリファイアウォール (WAF) インスペクションパイプラインおよび OWASP 保護
- [DDoS](/diagrams/ddos/) — DDoS 対策アーキテクチャおよびスクラビングサービス
- [API 保護](/diagrams/api-protection/) — API ゲートウェイセキュリティ、シャドー API 検出、レート制限
- [Bot 防御](/diagrams/bot-defense/) — Bot 検出パイプライン、クレデンシャルスタッフィング防御
- [ゼロトラスト](/diagrams/zero-trust/) — ZTNA アーキテクチャ、アイデンティティ検証、マイクロセグメンテーション

### インフラストラクチャ

- [DNS](/diagrams/dns/) — DNS 解決、GSLB、F5 XC DNS 管理
- [Kubernetes](/diagrams/kubernetes/) — Ingress コントローラー、サービスメッシュ、Pod ネットワーク
- [Terraform](/diagrams/terraform/) — HashiCorp Consul、Vault、および Nomad を使用した IaC 自動化

### F5 製品

- [F5 ブランド](/diagrams/f5-brand/) — F5 XC サービスアイコン、NGINX 製品ライン、BIG-IP アイコン
- [F5 Distributed Cloud](/diagrams/f5xc-diagrams/) — F5 ブランドアイコンを使用した XC WAAP、Bot 高度防御、マルチクラウドネットワーク、DDoS 対策、DNS、API セキュリティ
