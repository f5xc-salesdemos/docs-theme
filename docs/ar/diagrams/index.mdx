تتضمن سمة المستندات تكاملاً مع Mermaid مع حزم أيقونات مخصصة لبناء مخططات المعمارية. تصفح الفهرس أدناه للعثور على أمثلة لحالة الاستخدام الخاصة بك.

## البدء

راجع صفحة [التعليمات](/diagrams/instructions/) للاطلاع على مرجع الصياغة وحزم الأيقونات المتاحة وتفاصيل تنسيق CSS.

## أنواع المخططات

### الأساسيات

- [المخططات الأساسية](/diagrams/basic-diagrams/) — أمثلة على مخططات التدفق والتسلسل والدائرة والفئة والحالة وجانت

### مزودو السحابة

- [AWS](/diagrams/aws/) — معماريات VPC والشبكات الفرعية وALB وEC2 وEKS وLambda وS3
- [Azure](/diagrams/azure/) — VNet وبوابة التطبيقات ومجموعات مقياس الأجهزة الافتراضية وAKS وطوبولوجيا Hub-Spoke
- [GCP](/diagrams/gcp/) — VPC وGKE وCloud Functions وCloud SQL والموازن العالمي
- [الشبكات متعددة السحابات](/diagrams/multi-cloud/) — طوبولوجيا متعددة السحابات وموازنة التحميل العالمية واتصال F5 XC متعدد السحابات

### الأمان

- [WAF](/diagrams/waf/) — خط أنابيب فحص جدار حماية تطبيقات الويب (WAF) وحماية OWASP
- [DDoS](/diagrams/ddos/) — معمارية التخفيف من حماية DDoS وخدمات التنقية
- [حماية API](/diagrams/api-protection/) — أمان بوابة API واكتشاف API المخفي وتحديد المعدل
- [دفاع Bot](/diagrams/bot-defense/) — خط أنابيب كشف Bot والدفاع ضد حشو بيانات الاعتماد
- [الثقة الصفرية](/diagrams/zero-trust/) — معمارية ZTNA والتحقق من الهوية والتجزئة الدقيقة

### البنية التحتية

- [DNS](/diagrams/dns/) — تحليل DNS وGSLB وإدارة DNS لـ F5 XC
- [Kubernetes](/diagrams/kubernetes/) — وحدات تحكم Ingress وشبكة الخدمات وشبكات Pod
- [Terraform](/diagrams/terraform/) — أتمتة IaC مع HashiCorp Consul وVault وNomad

### منتجات F5

- [علامة F5 التجارية](/diagrams/f5-brand/) — أيقونات خدمة F5 XC وخط منتجات NGINX وأيقونات BIG-IP
- [F5 Distributed Cloud](/diagrams/f5xc-diagrams/) — XC WAAP ودفاع Bot والشبكات متعددة السحابات وحماية DDoS وDNS وأمان API باستخدام أيقونات علامة F5 التجارية
