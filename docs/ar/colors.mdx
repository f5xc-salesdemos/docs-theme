يُستخدم النطاق الكامل للوحة ألواننا في الرسومات والتوضيحات، مع استكشاف تركيبات فريدة وعصرية إلى جانب التباينات أحادية اللون. ويمكن توظيفها لإبراز موضوع محدد أو نقطة تركيز بعينها.

تستخدم لوحة F5 سبعة ألوان أساسية للعلامة التجارية بالإضافة إلى الأسود والأبيض، لكل منها أربع متغيرات من التدرجات الفاتحة والداكنة، بإجمالي **45 لوناً**. تُعدّ هذه الصفحة المرجع الوحيد الموثوق لتعيين ألوان العلامة التجارية إلى خصائص CSS المخصصة لـ Starlight أثناء تطوير السمات.

## التعيين إلى خصائص CSS المخصصة لـ Starlight

يعرض Starlight متغيرات `--sl-color-*` للتخصيص. فيما يلي التعيين من لوحة F5 إلى فتحات الألوان الدلالية في Starlight، المُنفَّذة في `styles/custom.css`.

> **كيفية عمل تخصيص سمات Starlight**: في Starlight، **الوضع الداكن هو الافتراضي لـ `:root`** — لا يوجد محدد `data-theme="dark"`. يستخدم الوضع الفاتح `:root[data-theme='light']`. متغيرات مقياس الرمادي *دلالية* وليست مرتبة حسب السطوع: `gray-1` هو دائماً "النص الأساسي" و`gray-6`/`gray-7` هما دائماً "الخلفيات الخفية"، بصرف النظر عن الوضع. تتبادل قيم السطوع الفعلية بين الوضعين. وبالمثل، `accent-high` هو دائماً لون نص التمييز عالي الوضوح و`accent-low` هو دائماً خلفية التمييز الخفية — يتعكس سطوعهما بين الوضعين.

```css
/* الوضع الداكن — الافتراضي لـ Starlight (:root) */
:root {
  --sl-color-white: #ffffff;      /* نص العناوين (الأكثر سطوعاً) */
  --sl-color-gray-1: #f5f5f5;    /* النص الأساسي — White المستوى 2 */
  --sl-color-gray-2: #cccccc;    /* نص الجسم — White المستوى 4 */
  --sl-color-gray-3: #999999;    /* نص خافت/مكتوم — Black المستوى 1 */
  --sl-color-gray-4: #666666;    /* الحدود — Black المستوى 2 */
  --sl-color-gray-5: #343434;    /* خلفية الكود المضمّن — Black المستوى 3 */
  --sl-color-gray-6: #222222;    /* خلفية الشريط الجانبي — Black المستوى 4 */
  --sl-color-black: #000000;     /* خلفية الصفحة */
  --sl-color-accent-low: #720016;   /* خلفية التمييز الخفية — F5 Red المستوى 4 */
  --sl-color-accent: #f06680;       /* الروابط، النشط — F5 Red المستوى 2 */
  --sl-color-accent-high: #f7b2bf;  /* نص التمييز — F5 Red المستوى 1 */
}

/* الوضع الفاتح — مقياس رمادي معكوس وسطوع تمييز معكوس */
:root[data-theme='light'] {
  --sl-color-white: #222222;      /* نص العناوين (الأكثر قتامة) — Black المستوى 4 */
  --sl-color-gray-1: #222222;    /* النص الأساسي — Black المستوى 4 */
  --sl-color-gray-2: #343434;    /* نص الجسم — Black المستوى 3 */
  --sl-color-gray-3: #666666;    /* نص خافت — Black المستوى 2 */
  --sl-color-gray-4: #999999;    /* الحدود — Black المستوى 1 */
  --sl-color-gray-5: #cccccc;    /* حدود فاتحة — White المستوى 4 */
  --sl-color-gray-6: #f5f5f5;    /* خلفية الشريط الجانبي — White المستوى 2 */
  --sl-color-gray-7: #faf9f7;    /* خلفية شريط التنقل — White المستوى 1 */
  --sl-color-black: #ffffff;     /* خلفية الصفحة */
  --sl-color-accent-low: #f7b2bf;   /* خلفية التمييز الخفية — F5 Red المستوى 1 (معكوس) */
  --sl-color-accent: #e4002b;       /* الروابط، النشط — F5 Red */
  --sl-color-accent-high: #720016;  /* نص التمييز — F5 Red المستوى 4 (معكوس) */
}
```

### التحقق من التباين (WCAG AA)

**الوضع الداكن** (نص على خلفية صفحة `#000000`):
- نص الجسم `#cccccc` → **16.3:1** (AAA)
- النص الخافت `#999999` → **10:1** (AAA)
- رابط التمييز `#f06680` → **5.6:1** (AA)

**الوضع الفاتح** (نص على خلفية صفحة `#ffffff`):
- نص الجسم `#343434` → **11.6:1** (AAA)
- النص الخافت `#666666` → **5.7:1** (AA)
- رابط التمييز `#e4002b` → **4.6:1** (AA)

### اقتراحات الألوان الدلالية

| متغير Starlight | لون F5 | حالة الاستخدام |
|--------------------|----------|----------|
| `--sl-color-accent` | F5 Red `#e4002b` | الروابط، الأزرار، الحالات النشطة |
| `--sl-color-text-accent` | F5 Red المستوى 2 `#f06680` | حالات التمرير في الوضع الداكن |
| `--sl-color-bg-accent` | F5 Red المستوى 4 `#720016` | خلفيات التمييز |
| مخصص: `--sl-color-info` | Bay `#0072b0` | تنبيهات المعلومات |
| مخصص: `--sl-color-success` | Jade `#009639` | رسائل النجاح |
| مخصص: `--sl-color-warning` | Tangerine `#f29a36` | تنبيهات التحذير |
| مخصص: `--sl-color-danger` | F5 Red `#e4002b` | تنبيهات الخطأ/الخطر |

## تباين الألوان

لضمان إمكانية الوصول وفهم اتصالاتنا للجميع، استخدم مجموعات ألوان الخلفية والنص التي تستوفي معايير إمكانية الوصول. اختر أزواجاً من الألوان بنسبة تباين لا تقل عن 3:1 للنص الكبير و4.5:1 للنص العادي.

يجب أن تستوفي جميع أزواج الألوان متطلبات تباين **WCAG AA**:

- **النص العادي** (أقل من 18pt / 14pt عريض): نسبة تباين لا تقل عن **4.5:1**
- **النص الكبير** (18pt أو أكبر / 14pt عريض): نسبة تباين لا تقل عن **3:1**
- **مكونات واجهة المستخدم والكائنات الرسومية**: نسبة تباين لا تقل عن **3:1**

### أزواج آمنة موصى بها

| المقدمة | الخلفية | نسبة التباين | النجاح |
|------------|------------|----------------|--------|
| White `#ffffff` | F5 Red `#e4002b` | 4.6:1 | AA |
| White `#ffffff` | River `#0e41aa` | 6.4:1 | AA / AAA |
| White `#ffffff` | Bay `#0072b0` | 4.6:1 | AA |
| White `#ffffff` | Jade `#009639` | 4.2:1 | AA (نص كبير) |
| White `#ffffff` | Eggplant `#62228b` | 6.5:1 | AA / AAA |
| White `#ffffff` | Raspberry `#ab2782` | 4.7:1 | AA |
| Black `#000000` | Tangerine `#f29a36` | 7.8:1 | AA / AAA |
| Black `#000000` | F5 Red المستوى 1 `#f7b2bf` | 10.1:1 | AA / AAA |
| Black `#222222` | White المستوى 1 `#faf9f7` | 15.8:1 | AA / AAA |

> **تلميح**: تحقق دائماً من نسب التباين باستخدام أداة مثل [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) عند إنشاء مجموعات ألوان جديدة.

## فحوصات السمة

- يستخدم حد `.swatch` القيمة `--sl-color-gray-5`
- يستخدم خلفية `.swatch-label` القيمة `--sl-color-gray-6`
- تعرض جميع عينات `--f5-*` لونها الفعلي (تُحلّ خصائص CSS المخصصة من `custom.css`)
- تظهر عينات الألوان في كلا وضعَي الفاتح والداكن
- تتحدث عينات رموز الشارات عند تبديل السمة
