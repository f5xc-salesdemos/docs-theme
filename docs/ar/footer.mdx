تستبدل السمة تذييل Starlight الافتراضي بمكوّن `Footer.astro` مخصص يضيف روابط وسائل التواصل الاجتماعي أسفل محتوى التذييل القياسي.

## نمط المكوّن

```astro
---
import Default from '@astrojs/starlight/components/Footer.astro';
---

<Default {...Astro.props}><slot /></Default>

<div class="social-links">
  <!-- social link icons -->
</div>
```

يقوم هذا النمط **بتوسيع** التذييل الافتراضي بدلاً من استبداله كلياً. يُصيّر `<Default {...Astro.props}><slot /></Default>` تذييل Starlight الأصلي (التنقل للأمام/الخلف)، ثم تُضاف روابط وسائل التواصل الاجتماعي أسفله.

## كيفية التسجيل

يُسجَّل تجاوز التذييل بواسطة مكوّن إضافي لـ Starlight في `index.ts`. أثناء خطّاف `config:setup`، يضبط تجاوز مكوّن `Footer`:

```ts
updateConfig({
  components: {
    ...config.components,
    Footer: '@f5-sales-demo/docs-theme/components/Footer.astro',
    // ... other component overrides
  },
});
```

لا تحتاج مستودعات المحتوى إلى تسجيل التذييل يدوياً — إذ يُطبَّق تلقائياً عند تحميل مكوّن السمة الإضافي.

| المنصة | الرابط | لون الأيقونة | تسمية Aria |
|--------|--------|--------------|------------|

| X | `https://x.com/F5` | `currentColor` (موروث) | X |
| LinkedIn | `https://www.linkedin.com/company/f5/` | `#0A66C2` (أزرق LinkedIn) | LinkedIn |
| Instagram | `https://www.instagram.com/f5.global/` | `#E4405F` (وردي Instagram) | Instagram |

يُفتح كل رابط في علامة تبويب جديدة (`target="_blank"`) مع `rel="noopener noreferrer"` لأغراض الأمان.

## تنسيق CSS

```css
.social-links {
  display: flex;
  justify-content: flex-end;
  align-items: center;
  gap: 1.25rem;
  padding: 1rem 0 0.5rem;
}

.social-links a {
  display: inline-flex;
  transition: opacity 0.2s ease;
}

.social-links a:hover {
  opacity: 0.7;
}
```

تُحاذى الروابط إلى اليمين في صف flex مع تأثير تعتيم خفيف عند التحويم.

## التخصيص

### تغيير رابط

عدّل سمة `href` على وسم `<a>` المقابل في `components/Footer.astro` داخل حزمة `@f5-sales-demo/docs-theme`.

### إضافة رابط

أضف عنصر `<a>` جديداً داخل div الخاص بـ `.social-links` مع أيقونة SVG بأبعاد 24×24. اتّبع النمط القائم:

```html
<a href="https://example.com" target="_blank" rel="noopener noreferrer" aria-label="Platform Name">
  <svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="#COLOR">
    <path d="..."/>
  </svg>
</a>
```

### حذف رابط

احذف كتلة `<a>...</a>` بأكملها الخاصة بالمنصة التي تريد إزالتها.

### تغيير ألوان الأيقونات

عدّل سمة `fill` على عنصر `<svg>`. استخدم `currentColor` لاستيراث لون النص، أو قيمة hex محددة لألوان العلامة التجارية.

### تغيير التخطيط

عدّل CSS الخاص بـ `.social-links` في كتلة `<style>`:
- **المحاذاة إلى المركز**: غيّر `justify-content` إلى `center`
- **تباعد أوسع**: زِد قيمة `gap`
- **المحاذاة إلى اليسار**: غيّر `justify-content` إلى `flex-start`
