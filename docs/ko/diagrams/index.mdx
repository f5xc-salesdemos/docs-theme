문서 테마에는 아키텍처 다이어그램을 구축하기 위한 커스텀 아이콘 팩이 포함된 Mermaid 통합이 포함되어 있습니다. 아래 카탈로그를 탐색하여 사용 사례에 맞는 예시를 찾아보세요.

## 시작하기

구문 참조, 사용 가능한 아이콘 팩 및 CSS 스타일링 세부 정보는 [지침](/diagrams/instructions/) 페이지를 참조하세요.

## 다이어그램 유형

### 기본 사항

- [기본 다이어그램](/diagrams/basic-diagrams/) — 플로우차트, 시퀀스, 파이, 클래스, 상태 및 간트 차트 예시

### 클라우드 공급자

- [AWS](/diagrams/aws/) — VPC, 서브넷, ALB, EC2, EKS, Lambda 및 S3 아키텍처
- [Azure](/diagrams/azure/) — VNet, 앱 게이트웨이, VM 스케일 셋, AKS 및 허브-스포크 토폴로지
- [GCP](/diagrams/gcp/) — VPC, GKE, 클라우드 함수, 클라우드 SQL 및 글로벌 LB
- [멀티클라우드](/diagrams/multi-cloud/) — 크로스 클라우드 토폴로지, 글로벌 부하 분산, F5 XC 멀티클라우드 연결

### 보안

- [WAF](/diagrams/waf/) — 웹 앱 방화벽 (WAF) 검사 파이프라인 및 OWASP 보호
- [DDoS](/diagrams/ddos/) — DDoS 보호 아키텍처 및 스크러빙 서비스
- [API 보호](/diagrams/api-protection/) — API 게이트웨이 보안, 섀도 API 검색, 속도 제한
- [Bot 방어](/diagrams/bot-defense/) — Bot 탐지 파이프라인, 자격 증명 탈취 방어
- [제로 트러스트](/diagrams/zero-trust/) — ZTNA 아키텍처, 신원 확인, 마이크로 세그멘테이션

### 인프라

- [DNS](/diagrams/dns/) — DNS 확인, GSLB, F5 XC DNS 관리
- [Kubernetes](/diagrams/kubernetes/) — 인그레스 컨트롤러, 서비스 메시, 파드 네트워킹
- [Terraform](/diagrams/terraform/) — HashiCorp Consul, Vault 및 Nomad를 활용한 IaC 자동화

### F5 제품

- [F5 브랜드](/diagrams/f5-brand/) — F5 XC 서비스 아이콘, NGINX 제품 라인, BIG-IP 아이콘
- [F5 분산 클라우드](/diagrams/f5xc-diagrams/) — F5 브랜드 아이콘을 사용한 XC WAAP, Bot 방어, 멀티클라우드 네트워킹, DDoS 보호, DNS, API 보안
