Das Dokumentationsthema enthält eine Mermaid-Integration mit benutzerdefinierten Symbolpaketen zur Erstellung von Architekturdiagrammen. Durchsuchen Sie den nachstehenden Katalog, um Beispiele für Ihren Anwendungsfall zu finden.

## Erste Schritte

Weitere Informationen zu Syntax, verfügbaren Symbolpaketen und CSS-Styling-Details finden Sie auf der Seite [Anweisungen](/diagrams/instructions/).

## Diagrammtypen

### Grundlagen

- [Grundlegende Diagramme](/diagrams/basic-diagrams/) — Beispiele für Flussdiagramme, Sequenz-, Kreis-, Klassen-, Zustands- und Gantt-Diagramme

### Cloud-Anbieter

- [AWS](/diagrams/aws/) — VPC-, Subnetz-, ALB-, EC2-, EKS-, Lambda- und S3-Architekturen

- [GCP](/diagrams/gcp/) — VPC, GKE, Cloud Functions, Cloud SQL und Global LB
- [Multi-Cloud](/diagrams/multi-cloud/) — Cloudübergreifende Topologie, globale Lastverteilung, F5 XC Multi-Cloud-Netzwerk

### Sicherheit

- [API-Schutz](/diagrams/api-protection/) — API-Gateway-Sicherheit, Shadow-API-Erkennung, Ratenbegrenzung
- [Bot-Abwehr](/diagrams/bot-defense/) — Bot-Erkennungspipeline, Credential-Stuffing-Abwehr
