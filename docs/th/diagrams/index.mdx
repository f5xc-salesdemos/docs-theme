ธีมเอกสารรวมการผสานรวม Mermaid พร้อมแพ็กเกจไอคอนแบบกำหนดเองสำหรับการสร้างแผนภาพสถาปัตยกรรม เรียกดูแคตตาล็อกด้านล่างเพื่อค้นหาตัวอย่างสำหรับกรณีการใช้งานของคุณ

## เริ่มต้นใช้งาน

ดูหน้า [คำแนะนำ](/diagrams/instructions/) สำหรับข้อมูลอ้างอิงไวยากรณ์ แพ็กเกจไอคอนที่มีให้ใช้งาน และรายละเอียดการจัดรูปแบบ CSS

## ประเภทแผนภาพ

### พื้นฐาน

- [แผนภาพพื้นฐาน](/diagrams/basic-diagrams/) — ตัวอย่างผังงาน ลำดับ วงกลม คลาส สถานะ และแผนภูมิแกนต์

### ผู้ให้บริการคลาวด์

- [AWS](/diagrams/aws/) — สถาปัตยกรรม VPC, subnets, ALB, EC2, EKS, Lambda และ S3
- [Azure](/diagrams/azure/) — VNet, App Gateway, VM Scale Sets, AKS และโทโพโลยี Hub-Spoke
- [GCP](/diagrams/gcp/) — VPC, GKE, Cloud Functions, Cloud SQL และ Global LB
- [เครือข่ายมัลติคลาวด์](/diagrams/multi-cloud/) — โทโพโลยีข้ามคลาวด์ การกระจายโหลดระดับโลก F5 XC multi-cloud connect

### ความปลอดภัย

- [WAF](/diagrams/waf/) — ไปป์ไลน์การตรวจสอบไฟร์วอลล์แอปเว็บ (WAF) และการป้องกัน OWASP
- [DDoS](/diagrams/ddos/) — สถาปัตยกรรมการบรรเทา DDoS และบริการกรองทราฟฟิก
- [การป้องกัน API](/diagrams/api-protection/) — ความปลอดภัย API gateway การค้นพบ shadow API การจำกัดอัตรา
- [การป้องกัน Bot](/diagrams/bot-defense/) — ไปป์ไลน์การตรวจจับ Bot การป้องกัน credential stuffing
- [Zero Trust](/diagrams/zero-trust/) — สถาปัตยกรรม ZTNA การยืนยันตัวตน micro-segmentation

### โครงสร้างพื้นฐาน

- [DNS](/diagrams/dns/) — การแก้ไข DNS, GSLB, การจัดการ DNS ของ F5 XC
- [Kubernetes](/diagrams/kubernetes/) — Ingress controllers, service mesh, เครือข่ายพ็อด
- [Terraform](/diagrams/terraform/) — การทำงานอัตโนมัติ IaC ด้วย HashiCorp Consul, Vault และ Nomad

### ผลิตภัณฑ์ F5

- [F5 Brand](/diagrams/f5-brand/) — ไอคอนบริการ F5 XC สายผลิตภัณฑ์ NGINX ไอคอน BIG-IP
- [F5 Distributed Cloud](/diagrams/f5xc-diagrams/) — XC WAAP, การป้องกัน bot, เครือข่ายมัลติคลาวด์, การป้องกัน DDoS, DNS, ความปลอดภัย API โดยใช้ไอคอนแบรนด์ F5
