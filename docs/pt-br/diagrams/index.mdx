O tema de documentação inclui uma integração com o Mermaid e pacotes de ícones personalizados para a construção de diagramas de arquitetura. Navegue pelo catálogo abaixo para encontrar exemplos para o seu caso de uso.

## Primeiros Passos

Consulte a página de [Instruções](/diagrams/instructions/) para referência de sintaxe, pacotes de ícones disponíveis e detalhes de estilização CSS.

## Tipos de Diagramas

### Fundamentos

- [Diagramas Básicos](/diagrams/basic-diagrams/) — Exemplos de fluxograma, sequência, pizza, classe, estado e gráfico de Gantt

### Provedores de Nuvem

- [AWS](/diagrams/aws/) — Arquiteturas de VPC, sub-redes, ALB, EC2, EKS, Lambda e S3
- [Azure](/diagrams/azure/) — VNet, Gateway de Aplicação, Conjuntos de Escala de VM, AKS e topologia Hub-Spoke
- [GCP](/diagrams/gcp/) — VPC, GKE, Cloud Functions, Cloud SQL e balanceamento de carga global
- [Multi-Cloud](/diagrams/multi-cloud/) — Topologia entre nuvens, balanceamento de carga global, F5 XC multi-cloud connect

### Segurança

- [WAF](/diagrams/waf/) — Pipeline de inspeção de firewall de aplicação web e proteção OWASP
- [DDoS](/diagrams/ddos/) — Arquitetura de mitigação de DDoS e serviços de limpeza de tráfego
- [Proteção de API](/diagrams/api-protection/) — Segurança de gateway de API, descoberta de shadow API, limitação de taxa
- [Defesa contra Bots](/diagrams/bot-defense/) — Pipeline de detecção de bots, defesa contra preenchimento de credenciais
- [Zero Trust](/diagrams/zero-trust/) — Arquitetura ZTNA, verificação de identidade, microssegmentação

### Infraestrutura

- [DNS](/diagrams/dns/) — Resolução de DNS, GSLB, gerenciamento de DNS do F5 XC
- [Kubernetes](/diagrams/kubernetes/) — Controladores de Ingress, malha de serviços, rede de pods
- [Terraform](/diagrams/terraform/) — Automação de IaC com HashiCorp Consul, Vault e Nomad

### Produtos F5

- [Marca F5](/diagrams/f5-brand/) — Ícones de serviços do F5 XC, linha de produtos NGINX, ícones do BIG-IP
- [F5 Distributed Cloud](/diagrams/f5xc-diagrams/) — XC WAAP, defesa contra bots, redes multi-cloud, DDoS, DNS e segurança de API usando ícones da marca F5
