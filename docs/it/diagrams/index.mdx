Il tema di documentazione include un'integrazione Mermaid con pacchetti di icone personalizzati per la creazione di diagrammi architetturali. Sfoglia il catalogo sottostante per trovare esempi adatti al tuo caso d'uso.

## Per iniziare

Consulta la pagina [Istruzioni](/diagrams/instructions/) per riferimenti sintattici, pacchetti di icone disponibili e dettagli sullo stile CSS.

## Tipi di diagrammi

### Fondamenti

- [Diagrammi di base](/diagrams/basic-diagrams/) — Esempi di diagrammi di flusso, sequenza, torta, classi, stati e diagrammi di Gantt

### Provider cloud

- [AWS](/diagrams/aws/) — Architetture VPC, subnet, ALB, EC2, EKS, Lambda e S3
- [Azure](/diagrams/azure/) — VNet, App Gateway, VM Scale Sets, AKS e topologia Hub-Spoke
- [GCP](/diagrams/gcp/) — VPC, GKE, Cloud Functions, Cloud SQL e bilanciamento del carico globale
- [Multi-Cloud](/diagrams/multi-cloud/) — Topologia cross-cloud, bilanciamento del carico globale, F5 XC multi-cloud connect

### Sicurezza

- [WAF](/diagrams/waf/) — Pipeline di ispezione del Firewall per applicazioni web (WAF) e protezione OWASP
- [DDoS](/diagrams/ddos/) — Architettura di mitigazione DDoS e servizi di scrubbing
- [Protezione API](/diagrams/api-protection/) — Sicurezza del gateway API, rilevamento di shadow API, limitazione della velocità
- [Difesa Bot](/diagrams/bot-defense/) — Pipeline di rilevamento bot, difesa contro il credential stuffing
- [Zero Trust](/diagrams/zero-trust/) — Architettura ZTNA, verifica dell'identità, micro-segmentazione

### Infrastruttura

- [DNS](/diagrams/dns/) — Risoluzione DNS, GSLB, gestione DNS F5 XC
- [Kubernetes](/diagrams/kubernetes/) — Controller Ingress, service mesh, rete dei pod
- [Terraform](/diagrams/terraform/) — Automazione IaC con HashiCorp Consul, Vault e Nomad

### Prodotti F5

- [Brand F5](/diagrams/f5-brand/) — Icone dei servizi F5 XC, linea di prodotti NGINX, icone BIG-IP
- [F5 Distributed Cloud](/diagrams/f5xc-diagrams/) — XC WAAP, difesa bot, Rete multi-cloud, Protezione DDoS, DNS, Sicurezza API con icone del brand F5
